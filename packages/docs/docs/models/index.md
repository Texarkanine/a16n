# Models

The **@a16njs/models** package defines shared TypeScript types and interfaces used across the a16n toolkit.

## Overview

This package provides:

- Common data structures for representing AI agent configurations
- Plugin interfaces for format-specific implementations
- Type guards and validation utilities
- Shared warning and error types
