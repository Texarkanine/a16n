# Plugin: Cursor

The **@a16njs/plugin-cursor** package implements the Cursor IDE format plugin for reading and writing `.cursor/rules/*.mdc` files and `.cursorignore` patterns.

## Overview

This plugin handles:

- Discovering Cursor rule files (`.mdc` format)
- Parsing MDC frontmatter and content
- Extracting file-specific rules, glob patterns, and metadata
- Emitting converted configurations back to Cursor format

## Supported Features

- ✅ Always-applied rules
- ✅ File-specific rules (glob patterns)
- ✅ Agent-requestable rules
- ✅ `.cursorignore` patterns
