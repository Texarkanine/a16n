# a16n Packages

This directory contains all packages in the a16n monorepo.

## Core Packages

- **[@a16njs/models](./models)** — Shared types, interfaces, and validation for the a16n ecosystem
- **[@a16njs/engine](./engine)** — Core conversion engine for translating between agent configuration formats
- **[a16n](./cli)** — Command-line interface for agent configuration conversion

## Plugins

- **[@a16njs/plugin-cursor](./plugin-cursor)** — Cursor IDE `.cursor/` format support
- **[@a16njs/plugin-claude](./plugin-claude)** — Claude Code `.claude/` format support  
- **[@a16njs/plugin-a16n](./plugin-a16n)** — a16n IR `.a16n/` intermediate representation format

## Utilities

- **[@a16njs/glob-hook](./glob-hook)** — Git hook for glob-based file rules (legacy, preserved for reference)
- **[docs](./docs)** — Documentation site source (Docusaurus)

For full documentation, see [https://texarkanine.github.io/a16n/](https://texarkanine.github.io/a16n/)
