# Glob Hook

The **@a16njs/glob-hook** package provides a glob pattern matching system for triggering actions based on file patterns in AI agent configurations.

## Overview

The glob-hook system enables:

- Matching file patterns against glob expressions
- Triggering hooks based on file changes or access
- Integrating with CI/CD workflows
- Enforcing file-specific rules
