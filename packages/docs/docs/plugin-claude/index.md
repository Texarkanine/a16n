# Plugin: Claude

The **@a16njs/plugin-claude** package implements the Claude Code format plugin for reading and writing `CLAUDE.md` files and `.claude/settings.json` ignore patterns.

## Overview

This plugin handles:

- Discovering Claude configuration files (`CLAUDE.md`)
- Parsing Claude markdown format and metadata
- Extracting ignore patterns from `.claude/settings.json`
- Emitting converted configurations back to Claude format

## Supported Features

- ✅ Nested `CLAUDE.md` files (directory hierarchy)
- ✅ Ignore patterns via `.claude/settings.json`
- ✅ Markdown section-based rule organization
